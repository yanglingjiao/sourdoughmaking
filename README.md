# 🍞 欧包计算器 - 专业酸面团配方计算工具

让每一份欧包都成为艺术品——基于科学算法与烘焙智慧的配方计算系统

---

## ✨ 核心功能

### 📊 智能配方计算
- **多种面粉类型支持**：T65、T55、黑麦面粉等欧洲标准面粉
- **精准含水量控制**：70%（经典型）/ 78%（平衡型）/ 85%（湿润型）三档选择
- **季节酵母策略**：自动调节酵母用量（夏季15% / 冬季25%）适应环境温度
- **Bassinage后加水**：支持传统法式后加水技术，优化面团筋力

### ⏰ 发酵时间管理
- **灵活发酵方案**：冷藏发酵（16-24小时）或常温发酵（3-5小时）
- **智能流程生成**：根据您的选择自动生成详细的揉面、发酵、烘烤时间表
- **实时状态追踪**：清晰展示每个阶段的当前状态和剩余时间

### 💡 专家级指导
- **烘焙技巧提示**：每个环节附有专业建议和注意事项
- **常见问题解答**：内置故障排查指南
- **配方保存**：支持保存常用配方，随时调用

## 📱 使用说明

### 快速开始
1. 选择面粉类型和目标含水量
2. 输入面团重量（克）
3. 根据季节选择酵母用量
4. 选择发酵方式（冷藏/常温）
5. 点击"生成配方"，获取完整制作流程

### 配方调整
- 支持实时修改参数，配方自动更新
- 所有配方比例经过专业面包师验证
- 内含温度、湿度等环境因素考量

## 🚀 部署方式

### 方法1：GitHub Pages（推荐免费）
1. 创建 GitHub 仓库并上传文件
2. 进入 Settings → Pages
3. Source 选择 "Deploy from a branch"
4. Branch 选择 main，文件夹 /(root)
5. 等待部署完成，获得访问链接

### 方法2：本地预览
```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js
npx serve

# 访问 http://localhost:8000
```

### 方法3：其他静态托管
- Vercel
- Netlify
- 腾讯云静态网站托管
