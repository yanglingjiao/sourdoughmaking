# 简单的PDF转换脚本

如果您的系统已安装Python，可以使用以下简单方法：

## 方法1：使用Pandoc（推荐）

### 安装Pandoc
1. 访问 https://pandoc.org/installing.html
2. 下载并安装Windows版本

### 转换命令
```bash
# 打开命令提示符，进入知识库目录
cd e:\烘焙书籍\欧包制作\知识库

# 转换单个文件
pandoc "Tartine bread (Chad Robertson) (z-library.sk, 1lib.sk, z-lib.sk).pdf" -o "..\知识库md\Tartine_Bread.md"

pandoc "The Perfect Loaf The Craft  (Maurizio Leo).pdf" -o "..\知识库md\The_Perfect_Loaf.md"

pandoc "如何驾驭开放式组织 Open Crumb Mastery (Trevor J. Wilson) (z-library.sk, 1lib.sk, z-lib.sk).pdf" -o "..\知识库md\Open_Crum_Mastery.md"
```

## 方法2：批量转换脚本

创建一个批处理文件 `convert_all.bat`：

```batch
@echo off
cd /d "e:\烘焙书籍\欧包制作\知识库"

echo 开始转换PDF文件...

pandoc "Tartine bread (Chad Robertson) (z-library.sk, 1lib.sk, z-lib.sk).pdf" -o "..\知识库md\Tartine_Bread.md"
if %errorlevel% equ 0 echo ✅ Tartine_Bread.md 转换成功

pandoc "The Perfect Loaf The Craft  (Maurizio Leo).pdf" -o "..\知识库md\The_Perfect_Loaf.md"
if %errorlevel% equ 0 echo ✅ The_Perfect_Loaf.md 转换成功

pandoc "如何驾驭开放式组织 Open Crumb Mastery (Trevor J. Wilson) (z-library.sk, 1lib.sk, z-lib.sk).pdf" -o "..\知识库md\Open_Crum_Mastery.md"
if %errorlevel% equ 0 echo ✅ Open_Crum_Mastery.md 转换成功

echo.
echo 转换完成！
pause
```

## 方法3：EPUB转Markdown

对于EPUB文件，使用Pandoc：

```bash
pandoc "The Perfect Loaf  The Craft and Science of Sourdough Breads, Sweets, and More  A Baking Book (Maurizio Leo).epub" -o "..\知识库md\The_Perfect_Loaf_Complete.md" -t markdown
```

## 📋 注意事项

1. **Pandoc安装后需要重启命令提示符**
2. **文件名中的特殊字符可能需要用引号括起来**
3. **转换后检查格式，可能需要手动调整**
4. **中文PDF转换效果可能不如英文PDF**

## 🚀 快速开始

1. 安装Pandoc
2. 复制上面的命令到命令提示符执行
3. 等待转换完成
4. 在 `知识库md` 文件夹中查看结果
